What is DSDB

DSDB is an acronym for Dynamic Segment Data Bank, which is a module that allows the programmist to use dynamicly sizeble memory bank, just as if he was using a file.

Reporting bugs

If you find bugs, please send it to my email, gynvael@lunarii.org. When reporting a bug, please try to follow this guidelines: 1. Give it a good title and specify that the bug is in the DSDB module. Titles like "Help me"or "another bug" are not very helpful when looking for a certain bug. Try to find titles like "DSDB: memory leak when cleaning pre allocated segments" or "DSDB: core dumped when trying to access last segment". 2. Send only one bug in one email. Makes it easier to handle and assign the bugs to different developers. 3. Try to use the correct spelling for keywords. 4. Please care for your reports! Sometimes the devs have questions. Without your answers we can't fix the bug. 5. See if your bug hasn't been fixed in the newer version of DSDB.

Big THANKS to people who care and send us bug reports!
