Set of scripts used to solve refrain task from 0CTF/TCTF quals 2019.

To generate `out/*` you have to use Ubuntu 16.04.6 server LTS with FreeType
and Courier font (aka `/usr/share/fonts/type1/texlive-fonts-recommended/pcrr8a.pfb`) installed.

You also need to use the example code from FreeType: <https://www.freetype.org/freetype2/docs/tutorial/example1.c>
